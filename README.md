Web Agency AI Lead Engine

A modular AI driven system designed to identify, analyse, score, and convert local web development opportunities.

Current Phase
Manual data collection with structured JSON input and Claude based opportunity scoring.

Core Workflow

Collect public business data

Structure data in lead_input_template.json

Run Local Opportunity Scanner

Store structured output

Track in leads.csv

Future Vision
Automated data ingestion from Google Maps and website APIs, structured analysis via Claude, and generation of actionable lead reports and outreach emails.
